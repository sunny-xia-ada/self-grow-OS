# 🎀 Loopy's Self-Grow OS (Loopy 岁末小记)

> **"Every loop is a growth, every entry is a treasure."**  
> 一个专为高审美用户打造的、基于 PWA 的隐私优先复盘系统。

![Loopy Icon](https://raw.githubusercontent.com/sunny-xia-ada/self-grow-OS/main/loopy_app_icon_1775255773227.png)

## 🌟 核心特性 (Key Features)

- **🚀 PWA (Progressive Web App)**: 支持“添加到主屏幕”，秒变独立 App。离线可用，响应迅速。
- **💾 IndexedDB 海量存储**: 突破浏览器 5MB 限制，轻松存储数千张 OOTD 高清图与万字长文。
- **📸 移动端原生体验**: 深度优化 iOS/Android 适配，支持直接调用相机捕获每一刻的灵感。
- **🔒 隐私第一**: 数据完全离线存储在您的设备中，不上传服务器，您的隐私由您掌控。
- **📑 岁末复盘系统**: 专为年末回顾设计的 sharding 存储逻辑，让每年的成长都有迹可循。

## 🛠️ 快速上手 (Quick Start)

### 作为 App 使用 (推荐)
1. 在手机浏览器（iOS Safari / Android Chrome）中打开项目部署链接。
2. 点击**分享/更多选项**。
3. 选择 **“添加到主屏幕” (Add to Home Screen)**。
4. 即可在桌面直接开启你的 Loopy 岁末小记。

### 自行开发/部署
1. 克隆本项目：`git clone https://github.com/sunny-xia-ada/self-grow-OS.git`
2. 使用任一静态服务器托管文件夹（如 Live Server, Vercel, 或 GitHub Pages）。
3. 访问 `index.html` 即可开始。
